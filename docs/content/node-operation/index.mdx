To set up a Flow node you will need to

1. [Provision](/node-operation/node-setup) the machine on which the node will run.

2. Generate the node identity by following the [bootstrap](/operator/node-bootstrap) process.

3. [Stake](/staking/locked-staking-guide) the node

4. And finally, [start](/operator/node-bootstrap#step-2---getting-ready-to-start-your-flow-node) the node
