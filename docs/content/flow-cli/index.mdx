The Flow CLI includes several commands to interact with Flow networks, such as querying account information,
or sending transactions. It also includes the [Flow Emulator](/emulator).

## Installation

Follow [these steps](https://github.com/dapperlabs/flow-cli#flow-cli) to install the Flow CLI on macOS, Linux, and Windows.
